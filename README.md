# 智能简历筛选工具

一个基于AI的智能简历筛选系统，支持简历自动解析、关键词匹配、标准化评估和协同流程管理。

## 功能特性

### 1. 简历上传与解析
- **多格式支持**：PDF、DOCX、TXT、JPG、PNG、BMP、TIFF
- **OCR识别**：支持图片格式简历的文本提取
- **智能提取**：自动提取姓名、联系方式、学历、工作年限、核心技能、工作经历等关键信息

### 2. 职位描述 (JD) 管理
- **JD创建**：HR可输入详细的招聘岗位要求
- **标签定义**：
  - 硬性指标：必须本科、必须有N年经验、必须来自特定公司、必须有特定职位
  - 加分项：加分技能、加分证书

### 3. 智能匹配与评分
- **匹配度评估**：系统根据简历内容与JD的契合度自动打分（0-100分）
- **多维度评分**：
  - 技能匹配度 (35%)
  - 经验匹配度 (25%)
  - 教育背景 (20%)
  - 证书资质 (10%)
  - 公司背景 (5%)
  - 职位匹配 (5%)
  - 简历完整度 (10%)
- **自动淘汰**：针对不符合硬性指标的简历自动归类

### 4. 协同与流程管理
- **简历看板**：Kanban形式展示"待筛选"、"待面试"、"已淘汰"状态
- **一键通知**：集成邮件功能，发送面试邀请或拒绝信
- **状态管理**：支持候选人在不同状态间的流转

## 技术栈

- **后端框架**：FastAPI
- **文档解析**：PyPDF2, python-docx
- **OCR识别**：pytesseract, OpenCV
- **文本处理**：jieba, nltk
- **机器学习**：scikit-learn
- **数据处理**：pandas, numpy

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动服务

```bash
python3 main.py
```

服务将在 http://localhost:8000 启动

### 2. 使用流程

1. **设置职位要求**：填写职位信息、硬性指标和加分项
2. **上传简历**：支持批量上传多种格式的简历
3. **查看解析结果**：系统自动提取简历信息
4. **开始评估**：系统自动匹配并生成评分
5. **管理流程**：在看板中管理候选人状态，发送通知

## 核心模块说明

### resume_parser.py
简历解析器，支持多种文件格式的文本提取和信息结构化。

### keyword_matcher.py
关键词匹配器，实现技能、教育、经验、证书等多维度匹配。

### evaluator.py
标准化评估器，提供多维度评分和等级评定。

### main.py
FastAPI Web服务，提供RESTful API和前端界面。

## API接口

### POST /api/job-requirements
设置职位要求

### POST /api/parse-resumes
解析上传的简历文件

### POST /api/evaluate
评估简历匹配度

### GET /api/job-requirements
获取当前职位要求

## 注意事项

1. **OCR功能**：需要安装tesseract-ocr才能使用图片识别功能
2. **邮件功能**：当前为模拟发送，实际使用需要配置邮件服务器
3. **文件上传**：上传的文件保存在uploads目录
